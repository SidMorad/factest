### Installation

`mvn compile`

`chmod +x organic.sh`

`./organic.sh`

Then see the failure.
Update `organic.sh` to point to correct FactSuite classpath(FSUIT variable).

`./organic.sh`
Then see the success.

### Run when changed

`npm i`

`npm run watch`
